# Thndr Task

## Deploy Demo

[Vercel Deploy](https://thndr-task-pink.vercel.app/)

## Start the project locally

- Clone ex. HTTPs `clone the repo`
- Run `yarn` to install dependencies in all packages
- Run `yarn start` (Host: 3000)
